# 安装 Zig (0.14.1 版本)
嘿，朋友！想玩转 Zig 是吧？这玩意儿安装比你想的简单，咱直接说重点。

Zig 的安装思路特直接：去官网下载个压缩包，解压，然后让你的电脑知道去哪儿找它就行。不像 Rust 有个 `rustup` 总管家，Zig 更像是“绿色软件”，解压就能用。

**第一步：去官网下载**

1.  **打开浏览器：** 访问 Zig 的官网 `ziglang.org`。
2.  **找到下载页：** 首页上应该有个大大的 "Download" 按钮，点进去。
3.  **选择版本：** 在下载页面，你会看到一堆版本。咱们要找的是 **0.14.1**。找到对应你电脑系统的版本下载。
    *   **Windows 用户：** 找名字里带 `windows-x86_64` 的 `.zip` 压缩包。
    *   **macOS (苹果电脑) 用户：**
        *   如果你用的是 Apple Silicon 芯片 (M1, M2, M3...)，找带 `macos-aarch64` 的。
        *   如果你用的是 Intel 芯片的老款 Mac，找带 `macos-x86_64` 的。
    *   **Linux 用户：** 找带 `linux-x86_64` 的。

**第二步：解压文件**

把下载下来的 `.zip` 文件解压。解压到哪儿都行，但最好找个你不会手滑删掉的地方。比如在 Windows 上可以放在 `C:\` 盘根目录，或者在 Mac/Linux 上放在你的用户主目录里。

解压后你会得到一个文件夹，名字类似 `zig-windows-x86_64-0.14.1` 这种，里面有个 `zig.exe` (Windows) 或者 `zig` (Mac/Linux) 文件，这就是主角。

**第三步：配置环境变量 (让电脑认识 zig 命令)**

这是最关键的一步，目的是让你在任何地方打开终端（黑窗口）都能直接用 `zig` 命令。

**在 Windows 上：**

1.  **复制路径：** 打开你刚才解压的那个文件夹，把地址栏里的路径整个复制下来。比如 `C:\zig-windows-x86_64-0.14.1`。
2.  **找到环境变量设置：**
    *   在桌面上右键点击“此电脑” -> “属性”。
    *   找到“高级系统设置”。
    *   在弹出的窗口里点“环境变量”。
3.  **修改 Path：**
    *   在“系统变量”或“用户变量”列表里找到叫 `Path` 的那一行，双击它。
    *   在弹出的窗口里点“新建”。
    *   把你刚才复制的文件夹路径粘贴进去。
    *   一路点“确定”保存。
4.  **重启终端：** 把所有打开的黑窗口（命令提示符或 PowerShell）都关了，再重新开一个，这样设置才能生效。

**在 macOS 或 Linux 上：**

1.  **打开终端 (Terminal)。**
2.  **编辑配置文件：** 你需要把你解压的 Zig 文件夹路径加到 shell 的配置文件里。
    *   如果你用的是 `bash` (老款 Mac 或一些 Linux 发行版)，就编辑 `~/.bash_profile` 或 `~/.bashrc`。
    *   如果你用的是 `zsh` (新款 Mac 默认)，就编辑 `~/.zshrc`。
3.  **添加命令：** 假设你把 Zig 解压到了 `~/zig-macos-aarch64-0.14.1`，就在配置文件的末尾加上这行：
    ```bash
    export PATH="$PATH:~/zig-macos-aarch64-0.14.1"
    ```
    **注意：** 把 `~/zig-macos-aarch64-0.14.1` 换成你自己的实际路径。
4.  **让配置生效：**
    *   运行 `source ~/.zshrc` (或者 `source ~/.bash_profile`)。
    *   或者干脆关掉终端再重开一个。

**最后一步：检查装好了没？**

1.  打开一个新的终端窗口。
2.  输入 `zig version` 然后敲回车。
3.  如果你看到了 `0.14.1`，恭喜你，大功告成！现在可以开始你的 Zig 之旅了！
