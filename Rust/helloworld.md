咱们来创建第一个 Rust 程序 "Hello, world!"，就像你对电脑说“你好，世界！”一样。 还是大白话哈：

**1. 创建一个文件夹（存放你的代码）：**

*   **随便找个地方：** 你可以在桌面、文档、或者任何你喜欢的地方创建一个文件夹，比如就叫 `rust_projects`。
*   **为啥要文件夹？** 文件夹就像你的小仓库，以后你的 Rust 代码文件就都放在这里。

**2. 创建一个 Rust 代码文件：**

*   **新建一个文件：** 在你刚才创建的文件夹里，新建一个文本文件。
*   **改文件名：** 把这个文本文件的名字改成 `main.rs`。  `.rs`  是 Rust 代码文件的后缀名。
*   **用记事本（或者别的编辑器）打开 `main.rs`**，把下面的代码复制进去：

```rust
fn main() {
    println!("Hello, world!");
}
```

*   **保存文件：**  把文件保存起来。

**代码解释（简单版）：**

*   `fn main() { ... }`  ：  这是 Rust 程序的“入口”，所有程序都是从这里开始执行的。
*   `println!("Hello, world!");`  ：  这个是 Rust 里的“打印”语句，它会在屏幕上显示 "Hello, world!"。

**3.  编译（把代码变成电脑能看懂的东西）：**

*   **打开终端（或者命令提示符/PowerShell）：**  还记得怎么打开吗？ Windows 上在开始菜单里搜，Mac 在“应用程序” -> “实用工具”里找。
*   **进入你的代码文件夹：**  用 `cd` 命令进入你刚刚创建的 `rust_projects` 文件夹。  例如，如果你的文件夹在桌面，可以这样输入 (按回车)：
    ```bash
    cd Desktop/rust_projects
    ```
    **重要：**  `cd`  后面要跟你的文件夹的**完整路径**。  如果不知道，可以在文件资源管理器/Finder里找到文件夹，然后右键点击文件夹，选择“属性”/“获取信息”，里面就有路径。
*   **编译代码：**  在终端里输入这个命令，然后按回车：
    ```bash
    rustc main.rs
    ```
    *   `rustc` 是 Rust 的编译器，它会把你的 `main.rs` 文件里的代码翻译成电脑能执行的“机器码”。
    *   如果成功编译，什么都不会显示，只是会多出一个文件，叫做 `main` (或者  `main.exe`  在 Windows 上)。 这就是你的可执行文件！

**4. 运行你的程序：**

*   **还是在终端里：**  确保你还在你的 `rust_projects` 文件夹里。
*   **运行命令：**  输入  `./main`  (或者  `.\main.exe`  在 Windows 上)  然后按回车。
    *   `./`  （Windows 上是 `.\`）表示“当前目录”。 
    *   `main` （或者 `main.exe`）是编译生成的程序文件。
*   **看结果：**  如果一切顺利，你会在终端里看到 "Hello, world!"  这几个字！

**总结：**

1.  **建文件夹:**  `rust_projects` (或者你喜欢的名字)
2.  **建代码文件:**  `main.rs`，里面写 `println!("Hello, world!");`
3.  **cd 进入文件夹:**  `cd /path/to/your/rust_projects`
4.  **编译:**  `rustc main.rs`
5.  **运行:**  `./main` (或者 `.\main.exe` 在 Windows 上)
6.  **看结果:**  终端里显示 "Hello, world!"

恭喜你，你的第一个 Rust 程序就跑起来啦！ 接下来就可以开始学习 Rust 语法，写更复杂的程序了。 棒棒哒！
