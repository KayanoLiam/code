# 安装 Rust
老铁！装 Rust 其实不复杂，咱说大白话：

不管是 Windows 还是苹果电脑 (macOS)，官方都推荐用一个叫 `rustup` 的小工具来装。把它想成是 Rust 的“总管家”，帮你安装、更新 Rust 都靠它。

**在 Windows 上装 Rust：**

1.  **去官网：** 打开你的浏览器，去 Rust 官网 `rust-lang.org`。
2.  **找下载：** 网站首页一般会有一个挺显眼的按钮，可能写着 “Get Started” 或者 “Install Rust” 之类的，点它。
3.  **下安装器：** 它会让你下载一个 `.exe` 文件，名字大概是 `rustup-init.exe`。把它下载到你电脑上。
4.  **运行安装器：** 双击你刚下载的那个 `.exe` 文件。
5.  **跟着提示走：**
    *   会弹出一个黑乎乎的窗口（命令行）。
    *   它会问你怎么装，通常有几个选项。一般直接按 `1`（默认安装）然后敲回车就行，它会帮你把需要的东西都装好。
    *   **注意：** 它可能会提示你需要装一个叫 “Visual Studio C++ build tools” 的东西。如果它提示了，最好按照它的指引去装一下，这个是后面写 Rust 代码编译时可能要用到的。如果没提示就不用管。
    *   等它跑完，显示安装成功。
6.  **重启终端（或者电脑）：** 装好之后，最好把你当前打开的所有黑窗口（命令提示符 或 PowerShell）都关了，再重新打开一个新的。有时候可能需要重启电脑才完全生效。

**在 macOS (苹果电脑) 上装 Rust：**

1.  **打开终端：** Mac 上装东西很多时候都用“终端”（Terminal）。你可以在“应用程序” -> “实用工具”里找到它，或者直接用 Spotlight 搜索 “终端” 打开。
2.  **输命令：** 在打开的终端窗口里，复制下面这行命令，粘贴进去，然后敲回车：
    ```bash
    curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
    ```
3.  **跟着提示走：**
    *   它也会问你怎么装，同样，一般直接敲回车（或者按 `1` 再敲回车）选择默认安装就行。
    *   它会自动下载、安装 `rustup` 和最新的稳定版 Rust。
    *   安装快结束时，它可能会提示你需要配置一下 `PATH` 环境变量，通常会告诉你运行 `source $HOME/.cargo/env` 这行命令，或者让你关闭当前终端窗口再重新打开一个新的。照做就行。

**装没装好怎么看？（Windows 和 macOS 都一样）**

1.  打开一个新的终端窗口（Windows 上是命令提示符 或 PowerShell，Mac 上是终端）。
2.  输入 `rustc --version` 然后敲回车。
3.  如果你看到类似 `rustc x.y.z (xxxxx yyyy-mm-dd)` 这样的版本号信息，那就说明你成功装上 Rust 啦！

**总结一下：**

*   都用 `rustup` 这个工具装。
*   Windows 下载 `.exe` 文件运行，跟着提示走（主要是选默认）。
*   Mac 在终端里运行一行命令，跟着提示走（主要是选默认）。
*   装完用 `rustc --version` 命令检查一下。

这样是不是清楚多了？祝你玩 Rust 玩得开心！
